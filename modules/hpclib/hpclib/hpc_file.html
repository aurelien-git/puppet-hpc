Puppet Defined Type: hpc_file

Defined in:
manifests/hpc_file.pp

Overview

Create a file with an hpc_source_file content

The source can be any source supported by the hpc_source_file function: - path - URL (http:// or file://) - Relative path are module path (like the template function)

Parameter Summary

  • source (Any) -

    Source(s) of the file (absolute path or URL), can be a a single value or an array of values

  • target (Any) => $title -

    Path of the file to write.

  • mode (Any) => '0644' -

    Permissions mode of the file.

  • owner (Any) => 'root' -

    User owner of the file.

  • group (Any) => 'root' -

    Group owner of the file.

  • backup (Any) => undef -

    Save the file before it is replaced.
