Overview

create a systemd unit file

A unit file is created, loaded in systemd and enabled

hpclib::systemd_service{ 'foo':
  target => '/etc/systemd/system/foo.service',
  config => {
    'Unit' => {
      'Description' => 'Fooing',
    },
    'Service' => {
      'Type'      => 'oneshot',
      'KillMode'  => 'process',
      'ExecStart' => '/usr/bin/foo',
    },
    'Install' => {
      'WantedBy' => 'multi-user.target',
    }
  }
}
