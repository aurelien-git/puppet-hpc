Overview

Configure pam_limits

The class will configure /etc/security/limits.conf and only on Debian activate the pam_limits module.

Parameter Summary

  • pam_service (Any) => $pam::limits::params::pam_service -

    Pam service name (default: 'common-session')

  • module (Any) => $pam::limits::params::module -

    Pam module name (default: 'pam_limits.so')

  • control (Any) => $pam::limits::params::control -

    Pam control (default: 'required')

  • type (Any) => $pam::limits::params::type -

    Pam type (default: 'session')

  • position (Any) => $pam::limits::params::position -

    Position of the module inside the file (default: after the comment "end of pam-auth-update config")

  • config_file (Any) => $pam::limits::params::config_file -

    Path of the configuration file for pam_access (default: `/etc/security/limits.conf`)

  • config_options (Any) => $pam::limits::params::config_options -

    Content of the pam_limits config file as a hash of key -> lines (default: {})
