Overview

Setup the OpenSSH server on the nodes with root keys

The module provides a default public root key that match the one setup by default in openssh::client, this should never be used in production. You must override it with the root_public_key parameter.

Note this is the actual key without the ssh-rsa and user name, not a file path.
