Overview

Install and configure clara. A Cluster administration tool.

Module Description

This module install clara and generate a usable configuration.

Setup Requirements

This module provides a default keyring, but it should be replaced by your own keyring.

Beginning with clara

If you wish to use the local repositories feature, you must provide a keyring. The example below, show how the default keyring was generated (the passphrase used is passphrase):
