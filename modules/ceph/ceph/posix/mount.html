Puppet Defined Type: mount

Defined in:
manifests/posix/mount.pp

Overview

Mount a POSIX Ceph FS

When using the kernel module, servers must be provided here. The FUSE mode uses the configuration in /etc/ceph/ceph.conf.

Parameter Summary

  • servers (Any) => undef -

    Array of Ceph MON servers (not supported in fuse mode)

  • key (Any) => undef -

    Name of the secret key file inside the keys directory

  • device (Any) => '/' -

    Directory to mount on the server

  • mountpoint (Any) => '/mnt' -

    Local mount point

  • user (Any) => 'admin' -

    Name of RADOS user

  • mode (Any) => 'kernel' -

    Kernel or FUSE mode (default: 'kernel')

  • ceph_file (Any) => '/etc/ceph/ceph.conf'
