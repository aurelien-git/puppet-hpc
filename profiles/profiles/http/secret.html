Overview

HTTP server for secrets

This http server listen on an alternate port (1216 with common.yaml). It serves an alternate document root and is protected by a firewall rules that will forbid connection not originating from a port strictly below 1024. This ensures the client is root.

This is used by the hpc-config-apply script.

Profiles dependency
